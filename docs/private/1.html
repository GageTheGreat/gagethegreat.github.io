AJ Discord Server Member Role


A new role has been introduced called Server Member. Server Member will have some perks such as:

- Being eligible for giveaways

- Being able to participate in game nights


In order to get the member role you:

  • Must have your nickname set to your real name
  • Must have your grade set
  • Must have talked in #main-chat at 10+ times
  • Must be online often (if you are invisible it will not count as online)
  • Must not have been warned, kicked or banned in the last 3 months

Rules:

  • If you break one of the server rules the member role will be taken away (temporarily)
  • If you stop following one of the member conditions for an exended period of time the member role will be taken away
