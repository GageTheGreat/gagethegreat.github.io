New Streaming Channel In AJ


You can now stream games to friends in the AJ discord server.

In order to be able to stream you need to take this form. If you pass you will get the streamer role.


Rules:

  • Do not interupt if someone else is streaming or else you will lose the ability to stream.
  • Only stream for a maximum of 3 hours.
  • If you are ever afk during stream for more than 15 minutes the stream will be shut down.
