Storage Editor

Commands:
'help' - gives you a help message
'get' - returns items of an item type
'edit' - edits an item
'del' - deletes an item
'create' - creates an item
'exit' - exits the app

Object types:
'cookie' - returns site cookies
'local' - returns localStorage items
'session' - returns sessionStorage items
